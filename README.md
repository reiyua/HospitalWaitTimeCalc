## 📌 About

HospitalWaitTimeCalc

Predicts how long a patient will wait based on appointment details.

---

## 🧠 Model

- Regression model  
- Output: Wait time (minutes)  

Inputs:
- Day  
- Time  
- Age  
- Gender  
- Doctor
